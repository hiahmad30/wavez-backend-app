# Wavez-Backend

### Installation

```bash
npm install
```

To run the server:

```bash
npm run dev
```

## Swagger Documentation:

[host]/api-docs (ex: http://localhost:3000/api-docs)

[host]/api-docs.json (ex: http://localhost:3000/api-docs.json | It can be validated on [https://editor.swagger.io](https://editor.swagger.io))
